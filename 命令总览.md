# ToolBox 命令总览

## e — 环境变量管理 + 路径打开器

### 基础操作
| 命令 | 说明 |
|------|------|
| `e` | 显示帮助信息 |
| `e <路径>` | 在资源管理器中打开路径 |
| `e -o <路径>` | 同上（显式 -o） |
| `e -g [模式]` | 打开环境变量对话框（user / admin / root） |
| `e -e` | 显示所有示例 |
| `e -v` | 显示版本 |

### 环境变量查看
| 命令 | 说明 |
|------|------|
| `e set` | 显示所有环境变量（带颜色） |
| `e set -l` | 左对齐变量名 |
| `e set -s` | 按分号逐行显示 |
| `e path` | 显示 PATH（带颜色） |
| `e -n` | 全局选项：禁用颜色输出 |

### Tag 配置管理
| 命令 | 说明 |
|------|------|
| `e tag list` | 列出所有 tag |
| `e tag create <名称>` | 创建新 tag |
| `e tag remove <名称>` | 删除 tag |
| `e tag edit <名称>` | 用记事本编辑 tag |
| `e tag -o` | 在资源管理器中打开 tags 目录 |

### Tag 配置编辑（增删改 PATH/变量/PROMPT/别名）
| 命令 | 说明 |
|------|------|
| `e tag path-add <tag> <路径>` | 添加 PATH 目录 |
| `e tag path-remove <tag> <索引>` | 按索引移除 PATH 目录 |
| `e tag var-set <tag> <名称> <值>` | 设置环境变量 |
| `e tag var-remove <tag> <名称>` | 移除环境变量 |
| `e tag prompt <tag> [值]` | 设置 PROMPT（留空清除） |
| `e tag alias-add <tag> <别名>` | 添加别名 |
| `e tag alias-remove <tag> <别名>` | 移除别名 |

### 脚本生成
| 命令 | 说明 |
|------|------|
| `e gen <tag1> <tag2> ...` | 组合 tag 生成 cmd 脚本 |
| `e gen --copy <tag1> <tag2> ...` | 生成并复制到剪贴板 |

### 配置管理
| 命令 | 说明 |
|------|------|
| `e config` | 显示配置目录路径 |
| `e config -o` | 在资源管理器中打开配置目录 |
| `e config --clear` | 清除配色配置，恢复默认 |

## as — 轻量级 Windows 软件包管理器

### 软件生命周期
| 命令 | 说明 |
|------|------|
| `as install <名称...>` | 安装软件 |
| `as list` | 列出可用软件及安装状态 |
| `as uninstall <名称...>` | 卸载软件 |
| `as upgrade [名称...]` | 升级已安装的软件 |

#### as install 选项
| 选项 | 说明 |
|------|------|
| `-v, --version <版本>` | 指定版本号 |
| `-g, --gui` | 使用图形界面向导安装 |
| `-r, --renew` | 强制重新下载 |
| `-d, --download-only` | 仅下载，不安装 |
| `--type portable\|installer` | 指定安装类型 |

#### as list 选项
| 选项 | 说明 |
|------|------|
| `-f, --filter <分类>` | 按分类过滤 |
| `-i, --installed` | 仅显示已安装 |
| `-m, --not-installed` | 仅显示未安装 |
| `-s, --search <关键字>` | 搜索名称、别名或描述 |
| `-d, --downloaded` | 仅显示已下载 |
| `--downloading` | 仅显示下载中 |
| `--no-download` | 仅显示未下载 |
| `-g, --group` | 按分类分组显示 |
| `--categories` | 显示所有分类概览 |
| `--info <名称>` | 查看软件详细信息 |

#### as uninstall 选项
| 选项 | 说明 |
|------|------|
| `-g, --gui` | 使用图形界面卸载 |
| `-f, --force` | 强制删除（跳过卸载器） |

#### as upgrade 选项
| 选项 | 说明 |
|------|------|
| `-c, --check` | 仅检查更新，不下载不安装 |
| `--renew` | 强制重新下载（即使版本相同） |

### 环境与配置管理
| 命令 | 说明 |
|------|------|
| `as config path` | 显示所有数据目录位置 |
| `as config path -o` | 在资源管理器中打开配置目录 |
| `as config cache` | 查看缓存文件列表 |
| `as config cache --clear` | 清除所有缓存 |
| `as config cache --open` | 打开缓存目录 |
| `as config source update` | 从远程仓库下载最新源定义 |
| `as config source path` | 显示源定义目录路径 |
| `as config source path -o` | 在资源管理器中打开源目录 |
| `as config speedtest [名称...]` | 测速所有下载源 |
| `as config speedtest -S` | 以软件为单位统计可用性 |
| `as config downloader list` | 列出所有下载后端及启用状态 |
| `as config downloader set <名称> on\|off` | 启用或禁用一个后端 |
| `as config downloader config` | 显示下载器配置文件路径 |
| `as config downloader config -o` | 在资源管理器中打开配置目录 |

### 自研工具管理
| 命令 | 说明 |
|------|------|
| `as tool install <名称...>` | 安装/更新自研工具 |
| `as tool upgrade [名称...]` | 升级自研工具 |
| `as tool list` | 列出可用自研工具及安装状态 |
| `as tool remove <名称>` | 移除自研工具 |

### as 自身管理
| 命令 | 说明 |
|------|------|
| `as self init` | 初始化 as 环境 |
| `as self update` | 更新 as 自身 |
